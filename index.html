Le projet consistait à développer la partie front-end du site d'une agence de location immobilière.

Le développement a été fait entièrement en React.

La homepage contient une liste des hébergements disponibles, sous la forme de vignettes contenant une photo.

L'utilisateur peut cliquer sur une vignette pour obtenir des informations détaillées sur le logement: localisation, information sur l'hôte, rating, galerie photo, équipement.

Ce projet était une occasion de s'initier à React, qui permet de créer facilement des applications web interactives.

Une des difficulté principales de ce projet était de permettre au visiteur de naviguer le site sans passer par le back-end: la fonctionnalité a été implémentée avec la librairie react-router.

Des composants réutilisables ont été créés pour le carousel (gallerie photo) et le collapse contenant la description des équipements.

Le projet consistait à développer la partie back-end du site web de notation de livres.

La homepage contient une liste des livres présent dans la base de données, avec une image et un rating.

L'utilisateur peut cliquer sur un livre pour accéder à la page spécifique décrivant le livre.

Un utilisateur peut créer un compte, se connecter et publier des livres sur le site.

Il est possible de modifier ses propres livres, et de noter tous les livres disponibles.

Ce projet était une occasion de pratiquer Node.js, un des languages les plus utilisés dans le dev back-end.

Une des difficulté principales de ce projet était d'implémenter l'authentification. Ici, nous avons choisi d'utiliser les JSON Web Token (JWT), qui permettent de garder les users authentifiés sans stockage de session coté serveur.

Une autre problématique était l'upload des images de couverture sur le serveur; nous avons utilisé la librairie multer, couplée à sharp, ce qui permet de compresser les images et optimiser le stockage.

About Me

Je suis un développeur web, fullstack.

Je m'efforce de proposer à mes clients des sites web avec une interface intuitive, une apparence moderne, responsive qui rendent la navigation agréable pour tous les utilisateurs, et mettent en valeur le contenu.

La performance d'un site web est un autre point fondamental pour moi; notamment, les temps de chargement doivent être réduits au minimum.

Je suis également sensible aux problématiques d'accessibilité et de référencement.

J'ai suivi la formation dev Web d'openclassrooms, qui m'a permis de mettre en pratique toutes les facettes du métier à l'occasion de plusieurs projets.

My Services

Web Design

Le Web design est l'art de concevoir et d'arranger du contenu dans un site web afin qu'il puisse être accessible online. Une combinaison d'esthétique et de fonctionnel, le web design détermine l'apparence de votre site web, comme ses couleurs, les polices de caractères, les images et l'interface utilisteur.

Le web design est un domaine en constante évolution, notamment les avec l'essor des différents modes de navigation (desktop, mobile, tablettes…). C'est une étape fondamentale dans la conception d'un site web percutant et créant une expérience captivante pour votre audience.

Web Development

Le développement d'un site Web comprend de nombreuses étapes, y compris l'acquisition d'un nom de domaine, le développement front-end et le développement back-end.

Le développeur est responsable de choisir les languages et technologies les plus adaptées, et les mettre en oeuvre pour concrétiser le projet. Il faut beaucoup de travail pour développer un site Web qui soit agréable à utiliser, sans temps de chargement, pour présenter votre marque sous un jour positif.

SEO

L'optimisation des moteurs de recherche (SEO) désigne la pratique consistant à adapter votre site Web afin qu'il bénéficie d'un meilleur classement sur une page de résultats de moteur de recherche dans le but d'augmenter le trafic sur votre site. L'objectif consiste à être classé sur la première page des résultats de Google pour les termes recherchés qui sont les plus pertinents du point de vue de votre audience cible.

Le SEO consiste donc principalement à comprendre les attentes et les besoins de votre audience tout autant que l'aspect technique de la configuration de votre site Web.
